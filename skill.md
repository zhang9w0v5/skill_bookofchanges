# 易经卜卦（Book of Changes）

## 概述

通过金钱卦（六爻）起卦，输出本卦、变卦、错卦、综卦、互卦、杂卦结果。

## 用法

### 命令行传参（AI调用）
```bash
python3 bookofchanges.py <卜卦之事>
```
自动起卦，结果保存到 `output` 子目录。

### 手动交互
```bash
python3 bookofchanges.py
```
输入卜卦之事后，可选择自动起卦或手动摇掷三枚硬币（每爻掷三次，共六爻）。

## 输出

终端打印起卦时间、卜事、卦象 JSON，同时 `output` 子目录生成 `{YYYYMMDD_HHMMSS}_results.md`。

## JSON 结构
```json
{
  "六爻": [...],
  "本卦": { "卦理": "...", "上卦": "...", "下卦": "...", "风火": "..." },
  "主变": { "六爻之和": "...", "天地余数": "...", "主变爻序": "...", ... },
  "变卦": { ... },
  "错卦": { ... },
  "综卦": { ... },
  "互卦": { ... },
  "杂卦": { ... }
}
```
