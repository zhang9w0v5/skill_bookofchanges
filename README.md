- 数字总和为 55

## 历法演进

### 上古历法推测

1. **夏朝太阳历**  
   现代考古推测：有些部落使用一年十个"月"的日历（太阳历），基于圆周数 360，每"月"36 日，与 365.2422 日相差 5.2422 日，长期使用不能吻合节气，无法准确指导农耕和计算年龄。

2. **女娲补天**  
   现代学术推测：女娲用五色石补天，补的是欠缺的五天，360 天十个月，五天用来过年和祭祀，共 365 天，但仍然有 0.2422 天的误差。

3. **绝地天通**  
   现代学术推测：颛顼帝为整顿民神杂糅不可方物，下令"绝地天通"——断绝民间与上天的直接沟通，将"观象授时"（观测天文、制定历法）的权力收归王室，由专职巫祝负责。

4. **天干地支的形成**  
   现代学术推测：
   - **羲和浴日**：十日一周，得十天干
   - **常羲浴月**：十二月一岁，得十二地支
   - 常羲与羲和同为帝俊之妻，天干地支而后形成六十甲子纪时体系（纪年、纪月、纪日、纪时）

5. **二十四节气的确立**  
   现代学术推测：夸父追日，是立杆追日影，确立冬至夏至，逐渐产生二十四节气。

6. **历法统一**  
   现代学术推测：后羿射日，十日射九日是指消灭不同历法，统一历法。

### 现代历法规则

**公历闰年规则**
- 平年 365 天，闰年 366 天
- 四年一闰，百年不闰，四百年再闰

**农历闰月规则**
- 农历也称阴阳历，结合了日月周期
- 19 年设置 7 个闰月，弥补农历年（约 354 天）与太阳回归年（约 365.24 天）的天数差
- 每个农历月以"朔日"（月球位于太阳和地球之间，肉眼看不到月亮）为初一
- 闰月放在不含"中气"（二十四节气中双数的节气，如雨水、春分）的月份之后，该月重复上一个月的名称（如闰四月）
- 二十四节气是农历的阳历成分，由太阳黄经位置确定，与月相无关

### 农历的本质

我们的农历是**阴阳历**，结合了太阳、月亮、地球运行规律的三维星历。视觉上北斗七星绕北极星旋转的规律，从而定义了二十四宿，进一步完善了天体坐标体系。

## 八卦易经

> 河图洛书、八卦符号先出现，后人著易经

### 易经的起源与演变

1. **先天八卦**  
   易经之始为伏羲所创先天八卦，后人称之为**乾坤易**，后有**连山易**和**归藏易**以及**梅花易**，相传由人类迁徙导致命名不同，是易经不同时期的名称。

2. **后天八卦**  
   由西周的周文王改造伏羲先天八卦而来：
   - 先天八卦主要表达地球时间
   - 后天八卦纳入了地球空间与人文
   - 代表着文明发生了巨大变化

3. **社会制度的变迁**
   - 先天八卦时期：禅位制
   - 后天八卦时期：世袭封建制

### 易经的意义

- **人文之祖**：易经是华人的人文之祖，我们的汉字和一些词汇均可追溯至易经
- **处事之道**：教化人们用发展的规律和不同的角度去思维
- **核心思想**：忧患意识，将忧患思维梳理清晰，避免毫无头绪的无效思维
- **实践方法**：遇事不决，则可以卜卦，通过卜卦来学习易经的思维，可厘清思维，化解无效的思维
- **卜卦本质**：八八六十四卦实际就是抽象的索引和字典，偶尔查查字典非常利于教化自身

## 金钱卦

### 基本规则

**使用工具**
- 常用乾隆通宝（民间认为乾隆在位超 60 年，流通人气足）
- 或现代人民币
- 字为阳，数字为 `[3]`
- 花或背面为阴，数字为 `[2]`

**爻的组成**
- 每爻由三个数字组成
- 共四种组合：
  - 三个阳 `[333]`
  - 三个阴 `[222]`
  - 两阳一阴 `[332]`
  - 两阴一阳 `[322]`

### 解卦方法

**阴阳难定时**
- 重点参考**错卦**（又称对卦）
- 错卦刚好是阴阳爻对立的卦
